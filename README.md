## Installation

### RubyGems

```
$ gem install fluent-plugin-mssql-lookup
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-mssql-lookup"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format filter mssql-lookup
```
